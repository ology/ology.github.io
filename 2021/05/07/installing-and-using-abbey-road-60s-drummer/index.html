Installing and Using Abbey Road 60s Drummer

Posted on 2021-05-07

Tags: music audio software

I had trouble installing and using Native Instruments "Abbey Road 60s Drummer" program. But after significant trial, error, and finally talking to NI support, this is the proper way to get it happening:

In short, you need to first install the Native Instrument program Kontakt 6 Player with the program Native Access. Then download and install the drummer app with it.

After that, running the Kontakt 6 Player will list the drummers, etc. in its library. Installing will also make the AU and VST plugins available for your DAW.

Whew. Simple in the end, but painful for a while there. ;)
