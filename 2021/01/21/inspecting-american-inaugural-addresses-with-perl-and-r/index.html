Inspecting American Inaugural Addresses with Perl and R

Posted on 2021-01-21

Tags: perl R software NLP analysis

Given all the inaugural addresses of American presidents, what are the readability stats?

As is often the case, I reach for perl to acquire and format data, and then explore it with R.

The code, fathom reads and analyzes a collection of text documents. It is based on the handy module Lingua::EN::Fathom.

So given my folder of inaugural addresses, I type this into the console:

perl fathom ~/Documents/lit/US/Inaugural-Addresses > ~/tmp/Inaugural-Address-stats.txt

And the tab-delimited, generated stats look like this:

...

Then in R, we read-in this data as so:

inaugural <- read.table('~/tmp/Inaugural-Address-stats.txt', header = TRUE, sep = "\t", quote = '')

Mostly, I just want to plot the measures over time to see if there is an upward or downward trend. Here is a function for this:

abplot <- function (data, title) {
    plot(data, type='l', main = title, xaxt = 'n')
    years <- seq(1789, 2021, by = 4)
    axis(1, 1 : length(years), years)
    abline(lsfit(1 : length(data), data))
}

Then we call this function for each column of data that we care about:

abplot(inaugural$Words, 'Number of Words')
abplot(inaugural$Sentences, 'Number of Sentences')
abplot(inaugural$WpS, 'Words per Sentence')
abplot(inaugural$FOG, 'FOG Readability')
abplot(inaugural$Flesch, 'Flesch Readability')
